О сервисе:

Проект «Treeography» — это уникальный онлайн сервис по посадке лесов. Он построен по принципу платформы c двумя группами пользователей: с одной стороны, лесничие, регистрирующие территории, на которых погиб лес; с другой стороны, люди, выбирающие национальный парк или заповедник, и количество деревьев для восстановления. Через сервис уже посажено 3 млн деревьев.
